About

The purpose of this web application is to help users organize tasks in an efficient way. Users will have the ability to enter their tasks for their list in the Add-Task tab. Once the user enters their data, it will be automatically sent to the dashboard and the labels pages. Within the dashboard tab, tasks will be ordered in the All-Tasks table, those that are due on the same day will automatically be displayed in the Due-Today tables. Users will have the ability to mark their tasks done by clicking them, while double-clicking will delete the task. The Labels page will output all the tasks organized by priority. There will be 3 dynamic tables in which data will be displayed according to the priority. The user will maintain the ability to mark tasks done or delete them as well.

This application was fully developed by me, Francisco Diaz de Villalvilla, and no one else. This project was created with the intention of improving readeability and code reutilization. If you have any ideas or proposals please contact me.
